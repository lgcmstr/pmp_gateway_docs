The American Society for Automation in Pharmacy (ASAP)

The Implementation Guide for ASAP Prescription Monitoring Program Web Service Standard can be downloaded from ASAP

Please note:

Appriss PMP Gateway service does not support the pick-list or the automatic polling functionality outlined in the Implementation Guide for ASAP Prescription Monitoring Program Web Service Standard. Appriss PMP Gateway service only supports the ad hoc query functionality.
